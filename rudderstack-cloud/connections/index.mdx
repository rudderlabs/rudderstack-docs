# Connections

RudderStack lets you set up connections to build efficient data pipelines across your entire customer data stack. You can add your preferred event data sources and destinations and establish connections between them to send your event data.

<img src="../assets/general/rudderstack-data-flow.png" alt="RudderStack data flow" />

## Setting up a connection

Setting up a connection in RudderStack involves the following steps:

### Step 1: Setting up a [**source**](#sources)

1. Log in to the **RudderStack dashboard**.

<img src="../assets/1 (26).png" alt="RudderStack Dashboard" />
<span class="imageTitle"></span>

2. Click **Sources** in the left navigation bar and then click on **New Source**. 

3. Click on the source you want to set up in RudderStack.

<img src="../assets/2 (30).png" alt="Add Source" />

4. Assign a name to the source, and click **Next**.
5. Configure the source with the relevant connection settings.

### Step 2: Setting up a [**destination**](#destinations) connected to source

1. Click **Add Destination** > **Create New Destination** from the **Source Details** window.

Alternatively, you can choose an existing destination by clicking on **Use Existing Destination**

2. Select the destination you want to configure with the source and click **Next**.

3. Configure the destination with the relevant connection settings.

You can view this connection in your RudderStack dashboard by clicking on the **Connections** tab on the left navigation bar, as shown:

<img src="../assets/general/rudderstack-connections.png" alt="RudderStack connections" />

You can also send your data from one source to multiple destinations, as shown:

<img src="../assets/general/rudderstack-onetomany-connection.png" alt="Multiple destinations for a single source" />
