# How to Integrate RudderStack with Your JAMstack Site

RudderStack supports integration with various popular JAMstack frameworks like Gatsby, Angular, Vue, Next.js, Hugo, Jekyll, and more. You can now seamlessly integrate the [RudderStack JavaScript SDK](https://www.rudderstack.com/docs/stream-sources/rudderstack-sdk-integration-guides/rudderstack-javascript-sdk/) into your web app/site built on these frameworks and set up your event streams in no time.
