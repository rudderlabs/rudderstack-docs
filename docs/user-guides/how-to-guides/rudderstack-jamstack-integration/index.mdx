RudderStack supports integration with various popular JAMstack frameworks like Gatsby, Angular, Vue, Next.js, Hugo, Jekyll, and more. You can now seamlessly integrate the <Link to="/sources/event-streams/sdks/rudderstack-javascript-sdk/">RudderStack JavaScript SDK</Link> into your web app/site built on these frameworks and set up your event streams in no time.
