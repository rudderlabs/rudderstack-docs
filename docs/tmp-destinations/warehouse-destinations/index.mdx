## Warehouse schema

When sending your events to a data warehouse via RudderStack, you don't need to define a schema for your event data - RudderStack automatically does that for you by following a predefined <Link to="/tmp-destinations/warehouse-destinations/warehouse-schema/">warehouse schema</Link>.
