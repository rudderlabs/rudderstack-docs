Destinations are the tools/platforms/applications where you can your event data via RudderStack.

RudderStack currently supports integration with over 150 popular destination platforms such as Google Analytics, Amplitude, Mixpanel, Salesforce, and more.

This section provides detailed documentation on setting up these event stream destinations in RudderStack to send your event data in no time.
