RudderStack Cloud is the advanced and comprehensive dashboard which helps you set up connections between your preferred event data sources and destinations and build efficient data pipelines across your entire customer data stack.

<div class="successBlock">
Sign up for the <a href="https://app.rudderstack.com/signup?type=freetrial" target="_blank">RudderStack Cloud (Dashboard)</a>.
</div>

You can bring together all your event data across your web, mobile, server-side, and cloud apps and route it reliably to your data warehouse and other third-party platforms of your choice.

Get started with RudderStack by setting up a source and connecting it to a cloud destination or data warehouse, as shown:
