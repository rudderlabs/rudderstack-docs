# macOS

[macOS](https://developer.apple.com/macos/) is Apple's operating system for the [Mac](https://www.apple.com/in/mac/) devices. It offers a SDK you can leverage to develop engaging and interactive macOS apps.

## iOS SDK support for macOS

Starting from version <strong>2.0.0</strong>, the <Link to="/stream-sources/rudderstack-sdk-integration-guides/rudderstack-ios-sdk/ios-v2/">RudderStack iOS SDK</Link> supports the macOS platform.

<div class="successBlock">

You can now integrate the iOS SDK with your macOS apps and seamlessly track user events without <strong>any</strong> additional configuration.
</div>

## Supported events

The macOS platform supports all the events that are supported by the <Link to="/stream-sources/rudderstack-sdk-integration-guides/rudderstack-ios-sdk/ios-v2/">RudderStack iOS SDK</Link>. These include:

* <Link to="/rudderstack-api/api-specification/rudderstack-spec/identify">Identify</Link>
* <Link to="/rudderstack-api/api-specification/rudderstack-spec/track">Track</Link>
* <Link to="/rudderstack-api/api-specification/rudderstack-spec/screen">Screen</Link>
* <Link to="/rudderstack-api/api-specification/rudderstack-spec/group">Group</Link>
* <Link to="/rudderstack-api/api-specification/rudderstack-spec/alias">Alias</Link>
* Reset
