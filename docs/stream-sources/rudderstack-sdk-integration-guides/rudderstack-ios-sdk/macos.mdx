# macOS

[macOS](https://developer.apple.com/macos/) is Apple's operating system for the mac devices. It offers a SDK you can leverage to develop engaging and interactive macOS apps.

## iOS SDK support for macOS

Starting from version <strong>2.0.0</strong>, the <Link to="/stream-sources/rudderstack-sdk-integration-guides/rudderstack-ios-sdk/">RudderStack iOS SDK</Link> supports the macOS platform.

<div class="successBlock">

You can now integrate the iOS SDK with your macOS apps and seamlessly track user events without <strong>any</strong> additional configuration.
</div>

## Supported events

The macOS platform supports all the events that are supported by the [iOS SDK](https://rudderstack.com/docs/stream-sources/rudderstack-sdk-integration-guides/rudderstack-ios-sdk/). These include:

* <Link to="/rudderstack-api/api-specification/rudderstack-spec/track">Identify</Link>
* <Link to="/rudderstack-api/api-specification/rudderstack-spec/track">Track</Link>
* <Link to="/rudderstack-api/api-specification/rudderstack-spec/track">Screen</Link>
* <Link to="/rudderstack-api/api-specification/rudderstack-spec/track">Group</Link>
* <Link to="/rudderstack-api/api-specification/rudderstack-spec/track">Alias</Link>
* <Link to="/rudderstack-api/api-specification/rudderstack-spec/track">Reset</Link>
