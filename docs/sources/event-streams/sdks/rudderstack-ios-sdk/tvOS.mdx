[**tvOS**](https://developer.apple.com/tvos/) is Apple's operating system for Apple TVs that are second generation or later. tvOS is based on the iOS operating system.

The tvOS platform also offers a tvOS SDK that you can leverage to develop apps that deliver engaging and interactive user experiences coupled with stunning picture quality and immersive sounds.

## iOS SDK support for tvOS

Starting from version <strong>1.1.0</strong>, the [**RudderStack iOS SDK**](https://rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/) supports the tvOS platform. 

<div class="successBlock">

You can now integrate the iOS SDK with your tvOS apps and seamlessly track user events without the need for <strong>any</strong> additional configuration.
</div>

## Supported events

The tvOS platform supports all the events that are supported by the [**iOS SDK**](https://rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/). These include:

* [**Track**](https://rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/#track)
* [**Identify**](https://rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/#identify)
* [**Screen**](https://rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/#screen)
* [**Group**](https://rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/#group)
* [**Alias**](https://rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/#alias)
* [**Reset**](https://rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/#reset)
