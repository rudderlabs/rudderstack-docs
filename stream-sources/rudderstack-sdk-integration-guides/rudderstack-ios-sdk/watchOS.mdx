# watchOS

[**watchOS**](https://developer.apple.com/watchos/) is the operating system developed for Apple Watch. It is based on the iOS operating system. watchOS also lets you leverage the watchOS SDK to develop apps that deliver interactive user experiences.

## iOS SDK support for watchOS

Starting from version <strong>1.3.0</strong>, the [**RudderStack iOS SDK**](https://rudderstack.com/docs/stream-sources/rudderstack-sdk-integration-guides/rudderstack-ios-sdk/) supports the watchOS platform. 

<div class="successBlock">

You can now integrate the iOS SDK with your watchOS apps and seamlessly track user events without the need for <strong>any</strong> additional configuration.
</div>

## Supported events

The watchOS platform supports all the events that are supported by the [**iOS SDK**](https://rudderstack.com/docs/stream-sources/rudderstack-sdk-integration-guides/rudderstack-ios-sdk/). These include:

* [**Track**](https://rudderstack.com/docs/stream-sources/rudderstack-sdk-integration-guides/rudderstack-ios-sdk/#track)
* [**Identify**](https://rudderstack.com/docs/stream-sources/rudderstack-sdk-integration-guides/rudderstack-ios-sdk/#identify)
* [**Screen**](https://rudderstack.com/docs/stream-sources/rudderstack-sdk-integration-guides/rudderstack-ios-sdk/#screen)
* [**Group**](https://rudderstack.com/docs/stream-sources/rudderstack-sdk-integration-guides/rudderstack-ios-sdk/#group)
* [**Alias**](https://rudderstack.com/docs/stream-sources/rudderstack-sdk-integration-guides/rudderstack-ios-sdk/#alias)
* [**Reset**](https://rudderstack.com/docs/stream-sources/rudderstack-sdk-integration-guides/rudderstack-ios-sdk/#reset)
