# RudderStack Features: Cloud vs. Open Source

This guide compares the feature differences between the two primary [RudderStack](https://rudderstack.com/) offerings - [RudderStack Cloud](https://app.rudderstack.com/signup?type=freetrial) and [RudderStack Open Source](https://app.rudderstack.com/signup?type=opensource).

<div class="infoBlock">

RudderStack also offers three plans as a part of <a href="https://app.rudderstack.com/signup?type=freetrial">RudderStack Cloud</a> : Free, Pro, and Enterprise. For detailed comparison and additional information on these plans, refer to the <a href="https://www.rudderstack.com/pricing/">Pricing</a> page.
</div>

## Feature comparison

| Feature |  Cloud |  Open Source | Reference | Comments |
| :--------| :-------------------| :--------------------------| :-----| :-----------|
| Data plane setup in your own environment |  ✖ |  ✓ | [Setup instructions](https://www.rudderstack.com/docs/get-started/installing-and-setting-up-rudderstack/) for RudderStack Open Source  | Cloud uses a RudderStack-hosted data plane. |
| Control plane setup in your own environment to manage sources and destinations |  ✖ |  ✓ | [Setup instructions](https://www.rudderstack.com/docs/get-started/control-plane-lite/) for RudderStack Open Source | Cloud uses a RudderStack-hosted [web app](https://app.rudderstack.com/) to manage to your connections. |
| Customize the data plane config parameters | ✖ |  ✓ |  [Configurable parameters](https://github.com/rudderlabs/rudder-server/blob/master/config/config.yaml) | - |
| Events backup |  ✓ |  ✓ | [Bucket configuration settings](https://www.rudderstack.com/docs/user-guides/administrators-guide/bucket-configuration-settings/) | - |
| Events replay | ✓ |  ✖ | [Documentation](https://www.rudderstack.com/docs/user-guides/administrators-guide/event-replay/) | This is an [Enterprise](https://www.rudderstack.com/enterprise-quote/)-only feature. | 
| Transformations |  ✓ |  ✖ | [Documentation](https://www.rudderstack.com/docs/transformations/) | Only applicable for [cloud mode](https://www.rudderstack.com/docs/connections/rudderstack-connection-modes/#cloud-mode) destinations. |
| Live events |  ✓ | ✖ | [Documentation](https://www.rudderstack.com/docs/get-started/live-events/) | Only applicable for [cloud mode](https://www.rudderstack.com/docs/connections/rudderstack-connection-modes/#cloud-mode) destinations. |
| Data governance | ✓ |  ✖ | [Documentation](https://www.rudderstack.com/docs/data-governance/)| This is an [Enterprise](https://www.rudderstack.com/enterprise-quote/)-only feature. |
| Cloud Extract (ETL) | ✓ |  ✖ | [Documentation](https://www.rudderstack.com/docs/cloud-extract-sources/) | [RudderStack Cloud Free](https://app.rudderstack.com/signup?type=freetrial) users can set up only 1 Cloud Extract source.  | 
| Reverse ETL | ✓ |  ✖ |  [Documentation](https://www.rudderstack.com/docs/warehouse-actions/) | [RudderStack Cloud Free](https://app.rudderstack.com/signup?type=freetrial) users can set up only 1 Warehouse Actions source.  |
