# RudderStack Docs

This repository contains the documentation markdown (`.mdx`) source files and the Gatsby code for [**rudderstack.com/docs**](https://rudderstack.com/docs/).

## About RudderStack

[**RudderStack**](https://rudderstack.com/) is the Customer Data Platform for developers. We provide efficient data pipelines that make it easy to collect data from any application, website or SaaS platform, and activate your data in a data warehouse and other business tools.

> Check out our [**Rudder Server repo**](https://github.com/rudderlabs/rudder-server) for more information on RudderStack.

## Contribute

See our [**contributing guide**](/CONTRIBUTING.mdx) for detailed instructions on how to build the docs and contribute/suggest improvements to RudderStack documentation. We accept all types of contributions - from quick fixes like typos, broken links, or updates, to more complex contributions like documentation for a new third-party integration that you want to propose.
