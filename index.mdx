# Home

[**RudderStack**](https://rudderstack.com/) is a **customer data platform (CDP) for developers.** It helps to deploy data pipelines which can collect customer data from every app, website and SaaS platform, apply any transformations if required and send it to various warehouses and downstream business tools.

<img src="assets/5 (20).png" alt="RudderStack basic functionality diagram" />

You can <a href="https://app.rudderstack.com/signup">sign up on RudderStack</a> and build a warehouse-first CDP that delivers complete, unified data to every part of your customer data stack.

## Key features
  - [**Event Streams**](https://rudderstack.com/docs/stream-sources/): Collects event data from all your sites and applications and route it to a wide array of customer tools and data warehouses.
  - [**Cloud Extract**](https://rudderstack.com/docs/cloud-extract-sources/): Builds efficient ETL pipelines to collect event data from different cloud tools and send it to your data warehouse.
  - [**Warehouse Actions**](https://rudderstack.com/docs/warehouse-actions/): Collects data residing in your warehouse and route the enriched information to desired destinations.
